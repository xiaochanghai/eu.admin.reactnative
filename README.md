## üëã Quick start

Clone the repo to your machine and install deps :

```sh
git clone https://github.com/xiaochanghai/eu.admin.reactnative

cd ./eu.admin.reactnative

pnpm install
```

To run the app on ios

```sh
npm run ios
```

To run the app on Android

```sh
npm run android
```
